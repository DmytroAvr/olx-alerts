# OLX Монітор

Я шукав конкретний товар на OLX і щодня перевіряв мої збережені пошуки в додатку у пошуках гарної пропозиції. Одного дня я знайшов чудову пропозицію, але вже було занадто пізно - товар був проданий.

Я побачив у цій ситуації можливість вивчити веб-скрейпінг за допомогою `nodejs`, щоб не пропустити наступну гарну пропозицію. Сподіваюся, що цей проект допоможе і вам.

## Встановлення та налаштування

Щоб використовувати цей скрипт, вам потрібно мати встановлені `node` та `npm`, мати обліковий запис на [Telegram](https://telegram.org/) та бажано комп'ютер, який працює цілодобово, щоб запустити скрипт.

Якщо ви вже знайомі з API Telegram і розумієте, як працюють боти, ось більш короткий посібник:

### Використання Node

1. Клонувати або завантажити репозиторій `git clone https://github.com/carmolim/olx-monitor.git`
1. Перейти в папку зі файлами js `cd src`
1. Встановити залежності командою `npm install`
1. Перейменувати файл `example.env` на `.env` і додати інформацію вашого BOT та групи, яка буде отримувати повідомлення
1. Додати URL-адреси, які ви хочете моніторити, у файлі `config.js`
1. Встановити інтервал перевірки в файлі `config.js`
1. Запустити скрипт командою `node index.js`
1. Стежити за ходом роботи скрипту в Терміналі
1. Якщо все пройшло вірно, у папці `data` будуть створені два нові файли: `ads.db` (база даних) та `scrapper.log` (логи виконання скрипту)

### Використання docker-compose

Якщо ви хочете використовувати Docker, щоб не встановлювати Node та залежності безпосередньо на ваш комп'ютер, виконайте такі кроки:

1. Виконайте кроки 1-7 з посібника для Node
2. Першого разу, коли ви запускаєте контейнер, потрібно побудувати образ командою `docker-compose build`
3. Наступні разі просто запустіть `docker-compose up`


### Налаштування Telegram

Щоб отримувати повідомлення через Telegram, вам потрібно мати деякі речі: бот з токеном та групу з цим ботом.

#### Створення вашого бота

Щоб отримати токен, вам потрібно створити власний бот. Я планую зробити подробний посібник, але поки що ви можете використовувати [цей](https://www.youtube.com/watch?v=4u9JQR0-Bgc&feature=[...]).

#### Знайдіть свій CHAT ID

Після створення вашого бота створіть групу та запросіть створений вами бот, а також інший бот `@idbot`, він допоможе вам дізнатися `CHAT_ID`, який нам потрібен для надсилання повідомлень.

Після додавання бота в групу просто введіть `/getgroupid@myidbot` і бот відповість з ID чату.

#### Редагування вашого середовища

У репозиторії є файл під назвою `example.env`, вам потрібно перейменувати його на `.env` і заповнити інформацію, яку ви щойно отримали.

| Змінна            | Приклад                                |
| ----------------- | -------------------------------------- |
| TELEGRAM_TOKEN    | Токен вашого бота від BotFather        |
| TELEGRAM_CHAT_ID  | ID вашого чату                         |

### Що потрібно моніторити?

Я не знаю, що ви шукаєте на OLX, але вам потрібно розповісти про це скрипту. Найпростіший спосіб - перейти на сайт OLX, виконати пошук, застосувати фільтри, які вам подобаються, та скопіювати URL.

Рекомендую використовувати досить специфічні фільтри, щоб не отримувати результати з дуже багатьма позиціями. Оскільки цей скрипт проходить через всі знайдені результати, можливо, що йому не вдасться обробити їх все.

Ви можете використовувати один або кілька пошуків, просто додайте `URL-адреси` у файл `config.js` у змінну `URLs`.

#### Приклади

##### Тільки одна `URL`

```
config.urls = ['https://sp.olx.com.br/sao-paulo-e-regiao/centro/celulares/iphone?cond=1&cond=2&pe=1600&ps=600&q=iphone']
```

##### Кілька `URLs`

Для використання кількох `URL-адрес` просто розділіть їх комою.

```
config.urls = [
    'https://sp.olx.com.br/sao-paulo-e-regiao/centro/celulares/iphone?cond=1&cond=2&pe=1600&ps=600&q=iphone',
    'https://sp.olx.com.br/sao-paulo-e-regiao/imoveis/venda?bae=2&bas=1&gsp=1&pe=600000&ps=100000&se=6&ss=2',
]
```

#### Порада

Чим конкретніше ваш пошук, тим ефективніше буде скрипт. Якщо ви просто шукаєте iPhone по всій Бразилії, ви будете отримувати багато повідомлень на день, це буде не дуже приємно.

## Як це працює

Робота скрипту дуже проста. Він проходить через масив `URL-адрес`, скопійованих з OLX, які вже містять фільтри мінімальної та максимальної ціни і т.д., знаходить оголошення на цій сторінці та зберігає їх у базі даних.

Записи, збережені в базі даних, потім використовуються для виявлення змін цін, які також повідомляються через Telegram.


## Важливі зауваження

- Цей скрипт працює тільки з бразильською версією OLX, в інших країнах інтерфейс інший і скрейпер не зможе отримати необхідну інформацію. Однак адаптація для інших сайтів можлива.
